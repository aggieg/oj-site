Older News

17th November 2014
Oddjob-1.4.0 is now available. Highlights include

  • New Icons and Job States to distinguish Jobs, Timers and Services.
  • Support for recursive directory listing with Ant style wild card expansion.
  • Repeat Job now iterates over any list and so has become a simpler Foreach Job.
  • Numerous Bug Fixes and Small Enhancements

More information can be found on the blog.

11th May 2013
Oddjob-1.3.0 is now available.

  • An Improved foreach Job that supports a parallel run window and the ability to design each job in the foreach configuration.
  • Timer and Retry now allow the nextDue property to be set while the timer is running.
  • A Grep Job has been added that can search files or any other input for text or a regular expression.
  • Numerous Bug Fixes and Small Enhancements

More information can be found on the blog.

12th September 2012
Oddjob-1.2.0 is now available. This release includes:

  • Improved Job Designer.
  • Improved Job Properties.
  • New JMX Support.
  • Numerous Bug Fixes and Small Enhancements

More information can be found on the blog.

27th January 2012
Oddjob-1.1.0 is now available. This release includes:

  • The old Struts Web Front End has been replaced with a new JSF one.
  • The ability to force a Failed Job to Complete has been added.

25th October 2011
Oddjob-1.0.0 is now available. This release includes:

  • Improved scheduling, more intuitive configuration and lots more examples.
  • Oddjob Explorer now has the ability to open new windows, navigation is easier and more accelerator keys have been added.
  • A read only role for the JMX client has been introduced and a JMX server can now use the Platform MBean.
  • Many bugs have been fixed, extra tests have been added, and job documentation improved.

3rd May 2011
Oddjob-0.30.0 is now available. This release includes:

  • Adoption of the international date standard ISO 8601 for all configuration.
  • An input handler for prompting for user input.
  • Improved Reference Guide including many more examples.

28th August 2010
Oddjob-0.29.0 is now available. This release includes:

  • Simple Archiving by taking a snapshot of a job's silhouette.
  • Support for distributing work across multiple servers by using a job that grabs for work.
  • Much improved SQL support, including running stored procedures and parameterised queries.

28th September 2009
Oddjob-0.28.0 is now available. This release includes:

  • Server Security with JMX Simple Security.
  • Network failure detection in the JMX client.
  • A cleaner core Oddjob achieved with more Oddballs.
  • A Simple services framework for auto wiring beans.

31st July 2009
Oddjob-0.27.0 is now available. This release includes:

  • A new module system for Oddjob called Oddballs. Includes an Oddball that lets Oddjob play with Spring.

17th May 2009
Oddjob-0.26.0 is now available. This release includes:

  • Cleaner server side Oddjob and improved user documentation for Sharing Oddjobs on the network.

8th April 2009
Oddjob-0.25.0 is now available. This release includes:

  • Simpler and more intuitive scheduling provided by three components: Timer, Retry and Trigger. These can be combined to meet most complicated scheduling requirements.
  • New state jobs for extra control over execution flow.

24th February 2009
Oddjob-0.24.0 is now available. This release includes:

  • New Java Bean based Configuration Framework that supports Drag and Drop, Job by Job, between Client and Server.

12th March 2006
Oddjob-0.22.0 is now available. This release includes:

  • New Improved web front end.
  • Improved persistence including database persistence.
  • New Simple SQL query job.
  • Extended support for java.lang.Runnable jobs.
  • Improved file/directory jobs.
  • Improved Developer Guide.
  • Lots More Examples.

7th December 2005
Oddjob-0.21.0 is now available. This release includes:

  • Right-click scheduling and unscheduling of any job on any server on any scheduler on any server.
  • Monitor a process's console output on any server.
  • Job scripting using groovy or other languages.
  • Cut and Paste support added to Oddjob Designer.
  • New User Guide.

5th October 2005
Oddjob-0.20.0 is now available. This release includes:

  • Quartz is now used for scheduling.
  • A schedule can now be triggered based on the completion state of another job.
  • Job properties can now be set directly from Oddjob Explorer.
  • System properties, environment variables and property files can now be used in Oddjob.
  • Documentation, particularly the reference guide has been improved.
